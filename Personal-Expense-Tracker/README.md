# Personal Expense Tracker

A Python based application to record, store, and analyze personal financial transactions.

## Features
- Add and store expenses with date, category, and description
- View complete transaction history
- Monthly and category wise expense analysis
- SQLite based persistent storage
- Modular and scalable backend design

## Tech Stack
- Python
- SQLite
- Pandas

## Why this project?
This project focuses on accurate financial data handling, analytics, and clean backend design , similar to systems used in finance and insurance domains.

## Future Enhancements
- Budget threshold alerts
- Expense trend analysis
- AI-powered natural language queries (RAG)
